This example uses a <audio> element with a control attribute. If you omit it, you will not see the audio player anymore! This is perfect for a game! Look at the functions play() and pause() called by the buttons below.



If you want to play short sounds that can occur very rapidly, streamed sound/music is not a good solution. Howler API allows you to download and decode sound samples in memory, and play them on demand. Whene the button becomes enabled the sound sample has been downloaded and decoded in memory. It can now be played. Click on the button to play this sound. Click it rapidly: you see, it's ok for a game!
