A shadow effect can be applied to the outlining rectangle of an element with the box-shadow CSS property. The box-shadow property is typically controlled with four values separated by spaces:
box-shadow: <x-offset> <y-offset> <blur> <color> ;
The offset values are dimension units (px, em, etc) can be positive or negative. Positive x values place the shadow to the right, and negative values place the shadow to the left. Similarly positive y values place the shadow vertically lower than the element and negative values move it up. The blur value is also a dimension unit, but can only be 0 or positive (sfocata).
