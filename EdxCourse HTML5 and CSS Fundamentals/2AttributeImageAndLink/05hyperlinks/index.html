hyperlink

Hyperlink is any text or image you can click and it will take you to another page. This page can be:
  1. Another Web page: Gazzetta dello sport
  2. A bookmark (a specific part of a Web page): e.g. the History section of the Hyperlinks wiki page at https://en.wikipedia.org/wiki/Hyperlink#History. We learned about using the id attribute as a link target in the unit about attributes earlier. href="#details" if we are moving on another part of the same page
  3. A local link (link to another part of the same Web page) relative URL
  4. An email: e.g. mailto:helloauthor@w3c.com Nevertheless, you should probably avoid using hyperlinks for email addresses: The email address usually opens up on the default email client on the user's computer which they might not use. This defeats the whole purpose. Providing your email address in your Web page directly as text or a hyperlink will expose your email address. Bots or spam crawlers can be used to search the Web to pick up email addresses for spam. You should avoid this if possible and use contact forms instead.
Appearance - links have a default appearance in most browsers, blue and underlined. Ensure no other text in your page is underlined to avoid confusing the user. They might get frustrated trying to click text that they think is a link.
